# Exploratory Data Analysis & Visualization: Russian Equipment Losses

This repository contains an end-to-end analysis of Russian military equipment losses during the conflict in Ukraine, utilizing data sourced from the WarSpotting.net API.

## Project Overview
The analysis explores how the nature of the conflict has evolved, focusing on:
* **Temporal Trends:** Changes in loss frequency and types over time.
* **Equipment Generation:** The shift from modern to older, Soviet-era equipment deployments.
* **Geospatial Analysis:** Mapping where and how losses occurred throughout different phases of the conflict.

## Repository Contents
* **Jupyter Notebook (.ipynb):** Contains the primary data processing, cleaning, and EDA engine.
* **Tableau Visualization Files:** Interactive dashboards mapping conflict intensity and loss trends.
* **Presentation (.pptx):** A summary of the project's methodology and core findings.

## Key Findings
* **Shift in Warfare:** The conflict has shifted from a multi-domain operation to a costly, attrition-heavy ground war.
* **Equipment Depletion:** There is a clear pattern of older generation equipment replacing modern stocks as the war progresses.
* **Tactical Evolution:** Russia utilizes older equipment in stalemate areas, while newer generation equipment is prioritized for offensive attempts.
* **Loss Status:** Data indicates a high equipment destruction rate, suggesting low survival rates for crews.

## Data Source
The dataset used in this analysis is from the [WarSpotting API](https://www.kaggle.com/datasets/zsoltlazar/equipment-losses-dataset-warspotting-api).
